未来罐 · 小红书摘录

在笔记详情页点击扩展图标，会把标题、正文与图片写入你指定的合集。服务端须已配置 PostgreSQL 与 腾讯云 COS（与网页版上传一致）。

与网页构建时的 VITE_API_BASE 相同，不要末尾斜杠。

多用户模式：在笔记网页已登录后，开发者工具 → Application → Local Storage → 键名 mikujar_admin_jwt。若前端启用了仅 httpOnly Cookie（VITE_AUTH_HTTPONLY_COOKIE），浏览器里不再有 JWT，需改用静态 API_TOKEN + 下方 userId，或暂时关闭 Cookie 模式做摘录。

使用静态 API_TOKEN 时，后端要求查询参数 userId；用个人 JWT 时可留空。

在网页侧栏对应合集上可通过开发者工具查看 data 属性，或从 GET /api/collections JSON 里复制 id。

与网页「笔记设置 → 新建笔记位置」保持一致：勾选表示插入到当前合集最前（与主站默认「顶部」相同）；若你在网页里选的是「底部」，请取消勾选。
